# Meme Generator
Create a meme that reflects your own sarcastic sense of humor 

# Deployed Version

https://laughoutloudgenerator.netlify.app/

## Technologies
Project is created with:
* React : version 18.2.0
* Vite : version 4.1.0

## Setup
To run this project, install it locally using npm:

```
$ cd ../memegenerator
$ npm install
$ npm run dev
```
